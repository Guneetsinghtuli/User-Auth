Welcome

  1. POST -> /api/v1/user/register ==> To Create a new user, add these things in the body
    • name
    • email
    • number
    • password

  2. POST -> /api/v1/user/login ==> To Login the account, add these things in the body
    • email
    • password

  3. POST -> /api/v1/user/details ==> To get the details of the specific account,To get the id of the user you must login first , add these things in the body
    • id

  4. POST -> /api/v1/user/login/pwd ==> To change password of the account, add these things in the body
    • oldPassword
    • newPassword

    Make sure to add a header authtoken you will get the auth token in the login API


  5. POST -> /api/v1/user/login/update ==> To update name or number of the user, add these things in the body
    • if you want to change the name use name
    • if you want to change the number use number
    • You can use both as well

    Make sure to add a header authtoken you will get the auth token in the login API
